HTML cheatsheet





HTML Tags


Tag Name Description
<!----> Comment Defines a comment.
<a> Anchor Defines a hyperlink.
<br> Break Defines a line break.
<img> Image Inserts an Image.
<center> Center Defines a center text.
<button> Button Defines a clickable button.
<code> Code Defines a computer code.



Attributes

Attribute Name Description
src Source Specifies the URL of the media file to play.
alt Alternative Text Specifies an alternative text for an area, if the img is not displayed.
autoplay Autoplay The audio or video should play as soon as possible.
border Border The border width.
class Class Defines a class.
id ID Defines an id.
